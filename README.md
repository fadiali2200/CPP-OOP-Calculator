# OOP-Based Calculator (C++)

## Description
This is a console-based calculator developed in C++ using Object-Oriented Programming principles. It handles basic arithmetic and includes error validation.

## Features
- **Encapsulation:** Used classes and private/public access specifiers.
- **Error Handling:** Prevents division by zero errors.
- **Precision:** Uses `double` for accurate decimal calculations.

## How to Run
1. Download the `.cpp` file.
2. Compile using any C++ compiler (e.g., Dev-C++, VS Code).
3. Run the executable.
